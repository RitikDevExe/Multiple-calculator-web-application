\# 🧮 Multi Calculator Web App



A simple and interactive web application that combines multiple useful calculators in one place. Built using HTML, CSS, and JavaScript.



\---



\## 🚀 Features



\- 📊 \*\*BMI Calculator\*\*

&#x20; - Calculates Body Mass Index using height and weight

&#x20; - Displays category (Underweight, Normal, Overweight, etc.)



\- 💰 \*\*Simple Interest Calculator\*\*

&#x20; - Calculates interest based on principal, rate, and time

&#x20; - Displays total amount



\- 🔥 \*\*Daily Calorie Calculator\*\*

&#x20; - Estimates daily calorie needs

&#x20; - Based on age, gender, weight, height, and activity level
